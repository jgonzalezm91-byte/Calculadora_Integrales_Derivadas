Descripción del proyecto



Esta página web es una calculadora interactiva diseñada para resolver operaciones de Cálculo Diferencial e Integral de forma clara, precisa y con una interfaz moderna.



Cuenta con un fondo tipo galaxia animado con partículas, lo que le da un toque visual atractivo sin distraer del contenido principal.







Funcionalidades principales



1\. Derivación simbólica  

 Calcula la derivada simbólica de la función ingresada usando `math.js`.



2\. Integración numérica  

&nbsp;  Realiza el cálculo de la integral definida entre dos límites mediante la Regla del Trapecio con 10,000 subdivisiones.



3\. Teclado rápido  

&nbsp;  Incluye botones para insertar funciones comunes (`sen`, `cos`, `tan`, `log`, `exp`, etc.), además de operadores y números.



4\. Interfaz moderna  

&nbsp;  - Fondo animado tipo galaxia con partículas.  

&nbsp;  - Diseño limpio con Tailwind CSS.  

&nbsp;  - Adaptable a distintos tamaños de pantalla.

Uso básico



1\. Abre el archivo `index.html` en cualquier navegador.  

2\. Escribe una función en el campo “Función f(x)” (ejemplo: `x^3 + cos(x)` o `1/x`).  

3\. Ajusta la variable si es necesario (por defecto es `x`).  

4\. Para derivar, haz clic en “Derivar Función”.  

5\. Para integrar, ingresa los límites y haz clic en “Calcular Integral Definida”.  

6\. El resultado aparecerá en la sección inferior.







Nota



\- La derivada se obtiene simbólicamente.  

\- La integral es una aproximación numérica.  

\- Se recomienda evitar funciones con discontinuidades en el intervalo de integración.
